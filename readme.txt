== Description ==

QQWorld收银台 初级版，为WooCommerce打造的QQWorld收银台初级版，仅支持桌面端的微信扫描二维码支付，需要更多支付方式如微信全平台支付、支付宝、银联支付、有赞支付等请购买专业版。

<strong>专业版：QQWorld收银台 / QQWorld Checkout</strong>

更多的支付方式。

<ul>
	<li>
		<p><strong>微信支付</strong></p>
		<ul>
			<li>需要签约</li>
			<li>支持PC端扫码支付</li>
			<li>支持微信浏览器支付（JSAPI）</li>
			<li>支持手机浏览器支付（H5）</li>
			<li>支持红包推广</li>
		</ul>
	</li>
	<li>
		<p><strong>支付宝</strong></p>
		<ul>
			<li>需要签约</li>
			<li>支持PC端支付</li>
			<li>支持手机浏览器H5支付</li>
		</ul>
	</li>
	<li>
		<p><strong>银联支付</strong></p>
		<ul>
			<li>需要签约</li>
			<li>支持桌面端支付（客户端需要证书）</li>
			<li>支持手机浏览器支付（无需证书）</li>
		</ul>
	</li>
	<li>
		<p><strong>有赞支付</strong></p>
		<ul>
			<li>免备案、免签约</li>
			<li>仅支持PC端支付宝和微信的扫码支付</li>
		</ul>
	</li>
</ul>

== Installation ==

1. 上传 `qqworld-checkout-lite` 目录到 `/wp-content/plugins/`
2. 在插件页启用 QQWorld-Checkout-lite 插件
